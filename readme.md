# Песня.

*Полевых цветов веночек, в утренней росе цветок,*

*Соловья запев свисточек, сок березовый – глоток.* 

*Тишины послушать вволю – по тропинке в лес густой,*

*Поболтать с березкой вдоволь про него и про любовь.*
